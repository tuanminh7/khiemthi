🧩 Module 1: Text → Speech (AI tự nhiên)
❗ Lưu ý:

Gemini KHÔNG phải TTS trực tiếp
→ bạn cần kết hợp:

👉 Giải pháp chuẩn:
Gemini → xử lý nội dung
TTS:
Google TTS / ElevenLabs (giọng xịn hơn)
💻 Flow:
Text/PDF → Gemini xử lý → tối ưu nội dung → TTS → audio
🧪 Code mẫu (Gemini + TTS)
import google.generativeai as genai
from gtts import gTTS

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-1.5-pro")

text = "Giải thích nội dung này đơn giản cho học sinh"
response = model.generate_content(text)

clean_text = response.text

tts = gTTS(clean_text, lang='vi')
tts.save("output.mp3")

📷 Module 3: Camera đọc chữ (OCR bằng Gemini)

👉 Đây là phần “xịn” nhất

💻 Code mẫu:
from PIL import Image
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-1.5-pro")

img = Image.open("image.jpg")

response = model.generate_content([
    "Đọc toàn bộ văn bản trong ảnh này",
    img
])

print(response.text)

👉 Sau đó:

response.text → TTS → đọc lên
